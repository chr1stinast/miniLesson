Effects

There are 4 different types of event methods:

  • Mouse Events
  • Keyboard Events
  • Form Events
  • Document/Window Events

All event methods share a similar syntax. The formula is basically:

$("element").click(function(){
  /// what happens when event is fired
});


Quick run-through of the most commmon event methods:

  • click()

    The click() method attaches an event handler function to an HTML element. The function is executed when the user clicks on the HTML element.

    Example:

    $("button").click(function(){
      $(this).hide();
    });

    When clicked, anything with the button tag will be hidden.

    This code also uses another event method, hide(). Hide (as it's name implies) just hides an element.
